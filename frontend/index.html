Core capabilities

Everything your Indian
warehouse needs

Built from the ground up for Indian ecommerce reality — volumetric pricing, courier zones, fragility, live inventory.

🧠
Hybrid AI Engine

Rule-based FFD bin packing + 5 ML models working together. Rules handle 95% of cases instantly. ML handles edge cases when rules cannot decide.

📦
Multi-Platform CSV Import

Auto-detects Shiprocket, Delhivery, Meesho, Amazon, Flipkart, Unicommerce CSV exports. No column renaming. Just upload and run.

💸
Volumetric Pricing Engine

Auto-calculates dim weight (L×W×H÷5000) vs actual weight. Picks chargeable weight. Applies zone-based courier rates precisely.

📊
Real-time Analytics

Live dashboard shows cost saved, efficiency score, box distribution, waste percentage — refreshed per order, not per day.

🏭
Live Box Inventory

Tracks box stock in real-time. Engine only selects in-stock boxes. Auto-deducts quantity per decision. Low-stock alerts built in.

🔶
Fragility Intelligence

Auto-detects fragile items from product category. Selects padded fragile-safe boxes only. Prevents damage claims before they happen.
